# 使用DashScope API的Python项目

这个项目演示了如何使用DashScope API进行各种AI功能，包括：

## 功能

1. **情感分析** - 使用大模型分析文本情感
2. **Function Call** - 使用函数调用功能进行天气查询
3. **表格读取** - 使用多模态模型提取表格图片内容

## 环境要求

- Python 3.8+
- dashscope

## 安装

```bash
pip install dashscope
```

## 使用方法

1. 设置环境变量 `DASHSCOPE_API_KEY` 为您的DashScope API密钥
2. 运行相应的Python脚本

## 文件说明

- `情感分析.py` - 情感分析示例
- `FunctionCall使用.py` - Function Call功能示例
- `表格读取.py` - 多模态表格读取示例
